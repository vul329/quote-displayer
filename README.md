# 佳句隨機顯示器 (Quote Random Displayer)

一個 Windows 桌面小工具，定時在螢幕角落顯示你收藏的佳句，支援系統托盤、自訂排程、主題切換與熱鍵收集。

## 功能

- **定時顯示** — 可設定間隔、隨機範圍或固定時間顯示佳句
- **系統托盤** — 縮小至托盤運行，右鍵選單控制所有功能
- **分類管理** — 自訂分類，選擇哪些分類參與輪播
- **權重模式** — 優先顯示較少出現的句子，避免重複
- **排程規則** — 多種規則可並存（間隔、固定時間、靜音時段）
- **主題切換** — 明暗兩種主題
- **全域熱鍵** — F8（可自訂）收集反白選取的文字，一鍵收藏
- **彈窗顯示** — 無邊框浮動卡片，支援淡入動畫
- **可攜式** — 資料存於執行檔目錄，隨身碟即可攜帶

## 下載

前往 [Releases](https://github.com/vul329/quote-displayer/releases) 下載最新版本。

- `quote-displayer.exe` — 可攜式執行檔，直接執行
- `佳句隨機顯示器_x64_zh-TW.msi` — 安裝程式

## 快速開始

1. 下載 `quote-displayer.exe` 放到任意目錄
2. 雙擊執行，系統托盤會出現圖示
3. 右鍵托盤圖示 → 編輯佳句 → 新增你的第一句佳句
4. 右鍵托盤圖示 → 選擇主題 → 挑選喜歡的樣式
5. 右鍵托盤圖示 → 顯示佳句，測試彈窗效果

### 收藏佳句

1. 在任何程式中反白選取文字
2. 按 `F8`（預設熱鍵），文字自動存入「未整理」分類
3. 可在編輯器中移動到其他分類

## 開發

```bash
# 安裝依賴
npm install

# 開發模式
npm run tauri dev

# 建置執行檔
npm run tauri build
```

## 技術棧

- **前端**: React 19 + TypeScript + Vite
- **後端**: Rust + Tauri v2
- **系統托盤**: tauri-plugin-tray-icon
- **資料儲存**: JSON 檔案
